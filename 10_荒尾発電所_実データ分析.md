```
SOCの変化で判断:
```

## 🔍 詳細分析：2024年10月19日のケース

### **観測されたデータ:**
```
日付: 2024年10月19日
計画値: -1778.3 kW（放電方向）
SOC変化: 100% → 10%
変化量: -90%（減少 = 放電）
```

### **容量計算:**

```python
発電所最大容量 = 2,000 kWh
SOC変化量 = 90%減少（放電）
理論的放電量 = 2,000 kWh × 90% = 1,800 kWh

実際の計画値 = 1,778.3 kW（絶対値）
効率損失 = 1,800 - 1,778.3 = 21.7 kWh
損失率 = 21.7 / 1,800 × 100% = 1.2%

→ 効率 98.8% ✅
```

### **この値の意味:**

```
基準値 = -1,778.3 kW（負の値 = 放電）

これは「放電計画値」を表す:
  平常時: 1,778.3 kW で放電中
  
上げ調整指令時:
  例: 放電を増やす
  1,778.3 → 1,900 kW (さらに放電)
  
下げ調整指令時:
  例: 放電を減らす
  1,778.3 → 1,000 kW (放電削減)
  1,778.3 → 0 kW (放電停止)
```

## 💡 基準値の符号に関する完全理解

### **従来の理解（一次調整力市場）:**
```
基準値 = 供出可能な調整力容量（絶対値）
例: 1,998 kW

これは「能力値」であり、常に正の数
平常時は 0 kW で待機
```

### **新しい理解（二次・三次調整力市場）:**
```
基準値 = 充放電計画値（符号付き）

基準値 > 0:
  例: +1,778 kW
  意味: 充電計画値
  動作: 平常時に充電している（SOC増加）
  市場: ネガワット型として参加（下げ調整）

基準値 < 0:
  例: -1,778 kW  ← 荒尾の実データ！
  意味: 放電計画値
  動作: 平常時に放電している（SOC減少）
  市場: ポジワット型として参加（上げ調整）

基準値 = 0:
  意味: 待機状態
  動作: 何もしない
  市場: ネガポジ型として中立位置
```

## 📊 関西電力の公式との整合性

### **関西電力特許（JP 7377392 B1）の公式:**

```python
基準値 B_Ref = (目標容量 - 現在容量) / 時間

荒尾の例（2024/10/19）:
  最大容量: 2,000 kWh
  現在SOC: 100% = 2,000 kWh
  目標SOC: 10% = 200 kWh
  時間: 仮に1時間とすると
  
  基準値 = (200 - 2,000) / 1
        = -1,800 kW（負 = 放電）
  
  実際の計画値: -1,778.3 kW
  差分: 21.7 kW（効率損失分）
  
  符号の意味:
    負（-）= 現在容量 > 目標容量
         = SOC減少方向
         = 放電 ✅
  
→ 公式と実データが完全に一致！ ✅
```
