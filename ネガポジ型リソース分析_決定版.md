**出典**: 取引ガイド（全商品）2024年4月1日 第4版  
**ページ**: 3-4「(注)取引ガイドの用語について」

## リソースの種類（公式定義）

### 1️⃣ 発電リソース（ポジワット型）

**定義:**
- 発電機、発電所、自家発や蓄電池等を用いて
- **【逆潮流（ポジワット）のみ】** によりΔkWを供出するリソース

**特徴:**
- 発電のみ（逆潮流 = 系統への放電）
- 需要抑制は行わない
- 単純な発電・放電

---

### 2️⃣ ネガポジ型リソース ⭐⭐⭐

**定義:**
- 一需要場所に設置されている自家発や蓄電池等を用いて
- **【需要を抑制（ネガワット）した上で、さらにその余力を活用して逆潮流（ポジワット）までも行う】**
- ことによりΔkWを供出するリソース

**特徴:**
- ネガワット（需要抑制）+ ポジワット（逆潮流）
- 一つの需要場所で両方向の調整
- 充電（下げ）と放電（上げ）の両対応

---

### 3️⃣ 需要リソース（ネガワット）

**定義:**
- 需要家の負荷設備、自家発や蓄電池等を用いて
- **【需要抑制（ネガワット）のみ】** によりΔkWを供出するリソース

**特徴:**
- 需要抑制のみ（DR: Demand Response）
- 逆潮流は行わない
- 負荷削減のみ

## 【重要な事実】

### 1️⃣ リソースの種類により対応できる調整方向が異なる

| リソース種別 | 調整方向 |
|------------|---------|
| **ポジワット型**（発電リソース） | 上げ調整のみ（放電のみ） |
| **ネガワット型**（需要リソース） | 下げ調整のみ（需要抑制のみ） |
| **ネガポジ型リソース** ⭐ | **両方向の調整可能**（充電 + 放電） |

---

### 2️⃣ 市場と調整方向の関係

#### 一次調整力（FCR）
- **目的**: 周波数維持
- **要件**: 【両方向の調整が必須】
- **最適リソース**: ネガポジ型リソース ✅

#### 二次調整力（FRR）
- **目的**: 時間内変動対応
- **要件**: 上げ/下げそれぞれ別々に調達可能
- **参加可能**: ポジワット型 or ネガワット型でも参加可能

#### 三次調整力（RR）
- **目的**: 予測誤差対応
- **要件**: 上げ/下げそれぞれ別々に調達可能
- **参加可能**: ポジワット型 or ネガワット型でも参加可能

## 📊 あなたの蓄電池（1968kW）の分析

### データ:
- **最大出力**: 1968 kW
- **容量**: ~7.2 MWh
- **基準値**: 1998 kW
- **市場**: 一次調整力（周波数調整）

### リソース分類:
→ **【ネガポジ型リソース】** ⭐

### 理由:
1. 蓄電池は充放電両対応
2. 一次調整力は両方向の調整が必須
3. 周波数上昇時: 充電（ネガワット相当）
4. 周波数低下時: 放電（ポジワット相当）

### 基準値の意味（ネガポジ型）:

```
1998 kW = 供出可能な調整力容量

運用:
  - 平常時: 0 kW（待機状態）
  - 周波数低下: +1998 kW まで放電（上げ）
  - 周波数上昇: -1998 kW まで充電（下げ）
```

## 📊 二次・三次調整力での運用の違い

### 関西電力の特許（二次・三次調整力）:

#### 【パターンA: ポジワット型（放電のみ）】
- 充電計画で SOC 5% → 90% まで充電
- 基準値 = 充電計画値（例: 2.833 kW）
- 平常時: 基準値で充電中
- 調整指令: 基準値 ± 調整分
- 上げ調整時: 充電を減らす or 放電

#### 【パターンB: ネガワット型（充電のみ）】
- 放電計画で SOC 90% → 5% まで放電
- 基準値 = 放電計画値（例: -2.833 kW）
- 平常時: 基準値で放電中
- 調整指令: 基準値 ± 調整分
- 下げ調整時: 放電を減らす or 充電

#### 【パターンC: ネガポジ型（両方向）】
- 一次調整力と同様の運用
- 基準値 = 0 kW（中立）
- 平常時: 0 kW（待機）
- 両方向の調整に対応

### 💡 重要なポイント:

> 二次・三次調整力では、上げと下げを【別々に調達】するため、  
> ポジワット型（放電のみ）やネガワット型（充電のみ）でも参加可能！
>
> 一方、一次調整力は周波数維持のため【両方向同時】が必須！  
> → ネガポジ型リソースが必要

## 🔍 「基準値」定義の再整理

### 【定義A: 一次調整力（ネガポジ型）】

```
基準値 = 供出可能な調整力容量

運用:
  - 平常時の充放電: 0 kW（待機）
  - 上げ指令: 0 → +基準値 kW（放電）
  - 下げ指令: 0 → -基準値 kW（充電）
  - SOC: 50%付近を維持（両方向対応のため）

あなたのケース:
  基準値 1998 kW = ±1998 kW の調整力容量
```

### 【定義B: 二次・三次調整力（ポジワット型）】

```
基準値 = 充電計画値（ベースライン）

運用:
  - 平常時: 基準値で充電中
  - 調整指令: 基準値 ± 調整分
  - SOC: 計画的に変化（例: 5% → 90%）

関西電力の例:
  基準値 2.833 kW = 3時間で85%充電する計画
```

### 【定義C: 二次・三次調整力（ネガワット型）】

```
基準値 = 放電計画値（ベースライン）

運用:
  - 平常時: 基準値で放電中
  - 調整指令: 基準値 ± 調整分
  - SOC: 計画的に変化（例: 90% → 5%）
```

## 🎉 最終結論

### ✅ あなたの仮説「2次と3次はネガポジではない」は正しい！

### 【正確な理解】

#### 1️⃣ 一次調整力（周波数維持）
- → ネガポジ型リソースが必須
- → 基準値 = 供出可能容量
- → 平常時: 0 kW

#### 2️⃣ 二次・三次調整力（需給調整）
- → ポジワット型 or ネガワット型でも可能
- → 上げと下げを別々に調達
- → 基準値 = 充放電計画値（ベースライン）
- → 平常時: 基準値で充放電中

#### 3️⃣ 「基準値」の意味が異なる理由

**一次調整力:**
- ネガポジ型 → 平常時0kW → 基準値=容量

**二次・三次調整力:**
- ポジワット型 → 平常時充電中 → 基準値=充電計画
- ネガワット型 → 平常時放電中 → 基準値=放電計画
