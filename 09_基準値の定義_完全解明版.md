**作成日**: 2025年11月6日  
**内容**: 一次調整力 vs 二次・三次調整力における「基準値」の意味の違いを完全解明

---

## 🎯 核心的な発見

### **「基準値」という同じ言葉が、市場により全く異なる意味を持つ！**

| 市場 | 基準値の意味 | 値の例 | 符号 | 平常時の動作 |
|------|------------|-------|------|------------|
| **一次調整力** | 供出可能な調整力**容量** | 1998 kW | 絶対値 | **0 kW**（待機） |
| **二次・三次調整力** | 充放電**計画値** | +1998 kW | **常に正（+）** | **基準値で充放電中** |

## 📊 一次調整力の基準値（あなたの会社のケース）

### **市場特性:**
- **目的**: 周波数維持（50Hz/60Hz）
- **応動時間**: 10秒以内
- **リソース**: ネガポジ型**必須**（両方向調整が必要）

### **基準値の定義:**

```
基準値 = 供出可能な調整力容量
      = 1998 kW（片方向の容量）

これは「能力値」を表す
```

### **運用:**

```
平常時:
  充放電電力 = 0 kW（待機状態）
  SOC = 50%付近を維持

周波数低下時（上げ調整）:
  0 kW → +1998 kW（放電）
  
周波数上昇時（下げ調整）:
  0 kW → +1998 kW（充電）※消費として
  
供出可能な調整幅:
  上げ: 1998 kW
  下げ: 1998 kW
  合計柔軟性: 3996 kW相当
```

### **重要ポイント:**

✅ 基準値 = **容量**（能力値）  
✅ 平常時は **0 kW**（基準値を使わない）  
✅ 指令時に初めて **±基準値**の範囲で動作  
✅ SOCは常に **50%付近**（両方向対応のため）

## 📊 二次・三次調整力の基準値（関西電力の特許）

### **市場特性:**
- **目的**: 需給調整、予測誤差対応
- **応動時間**: 15〜45分以内
- **リソース**: ポジワット型、ネガワット型、ネガポジ型**すべてOK**

### **基準値の定義:**

```
基準値 = 充放電計画値（ベースライン電力）
      = (目標容量 - 現在容量) / 時間
      = 常に正の値（+）

これは「計画電力値」を表す
```

### **関西電力特許の公式（JP 7377392 B1 [0082]）:**

```python
基準値 B_Ref = (目標容量 - 現在容量) / 時間

例: 10kWh蓄電池
  現在SOC: 5% (0.5 kWh)
  目標SOC: 90% (9.0 kWh)
  時間: 3時間
  
  基準値 = (9.0 - 0.5) / 3 = +2.833 kW（充電計画値）
```

### **あなたの蓄電池（~7.2MWh）の場合:**

```python
容量: 7,200 kWh
現在SOC: 5% (360 kWh)
目標SOC: 90% (6,480 kWh)
時間: 3時間

理論値:
  基準値 = (6,480 - 360) / 3 = +2,040 kW

最大充電電力の制約:
  実際の基準値 ≤ +1,998 kW
  
→ 基準値 = +1,998 kW（充電計画値）
```

### **運用（ポジワット型として参加）:**

```
平常時:
  充電電力 = +1,998 kW（基準値で充電中）
  SOC: 5% → 90% へ計画的に変化

上げ調整指令時:
  充電を減らす or 放電
  例: 1,998 - 500 = 1,498 kW 充電
  例: 1,998 - 2,500 = -502 kW（放電に転換）

下げ調整指令時:
  充電を増やす（最大充電電力まで）
  例: 1,998 + 0 = 1,998 kW（既に最大なので増やせない）
```

### **重要ポイント:**

✅ 基準値 = **充放電計画値**（ベースライン電力）  
✅ **常に正の値（+）** で表記  
✅ 平常時から **基準値で充放電中**  
✅ 調整指令時は **「基準値 ± 調整分」**  
✅ SOCは **計画的に大きく変化**（5% → 90%など）

## 🔍 符号と方向の正しい理解

### **一般的な符号規則:**

| 動作 | 蓄電池の動作 | 系統への影響 | 調整方向 | 市場での表記 |
|------|------------|------------|---------|------------|
| **放電** | 放電（逆潮流） | 電力供給 | **上げ** | **+** (正の電力) |
| **充電** | 充電（順潮流） | 電力消費 | **下げ** | **+** (正の電力) |

**重要:** 市場での「基準値」は**電力の大きさ**として常に正の値で表記し、方向（充電/放電）は別途指定

## 💡 重要な洞察

### **1️⃣ なぜ同じ「基準値」という言葉なのに意味が違うのか？**

**一次調整力（ネガポジ型）:**
- 周波数維持 = 瞬時の両方向調整が必須
- 平常時は何もしない（待機）
- 基準値 = 「±この範囲で調整できる」という**容量**

**二次・三次調整力（計画的運用）:**
- 需給調整 = 計画的な充放電が前提
- 平常時から充放電している
- 基準値 = 「この電力で充放電する」という**計画値**

→ **市場の性質が根本的に異なるため、基準値の意味も異なる**

---

### **2️⃣ ネガポジ型の供出可能量について**

```
あなたの質問:
「ネガポジだと供出可能量 = 基準値1998 + 発電上限1998 = 3996？」

正しい理解:
  基準値（入札値）= 1998 kW（片方向）
  
  実際の供出能力:
    上げ調整能力: 1998 kW
    下げ調整能力: 1998 kW
    ───────────────────
    合計の柔軟性: 3996 kW相当 ✅

注意:
  「基準値 = 3996 kW」ではない
  「基準値 = 1998 kW」で入札
  「供出能力 = 両方向で合計3996kW相当」
```

---

### **3️⃣ 関西電力の基準値は「ただの充放電計画値（常に+）」**

✅ **完璧に正しい理解！**

```
関西電力特許の基準値:
  = 充放電計画値
  = ベースライン電力
  = 常に正の値（+）
  = 平常時にこの値で運用
  = 対象は二次・三次調整力市場
```

## 🎉 完全解明された内容

### **確認された事実:**

1. ✅ **「基準値」は市場により意味が異なる**
   - 一次調整力: 供出可能な調整力容量
   - 二次・三次調整力: 充放電計画値

2. ✅ **一次調整力はネガポジ型必須**
   - 周波数維持は両方向同時調整が必要
   - 平常時0kW、SOC 50%維持

3. ✅ **二次・三次調整力は片方向でもOK**
   - ポジワット型、ネガワット型でも参加可能
   - ネガポジ型はもちろん参加可能（最も柔軟）

4. ✅ **関西電力の基準値 = 充放電計画値（常に+）**
   - 平常時もこの値で充放電中
   - 調整時は「基準値 ± 調整分」

5. ✅ **ネガポジ型の供出能力 = 3996kW相当**
   - 上げ: 1998 kW
   - 下げ: 1998 kW
   - 合計の柔軟性: 3996 kW

6. ✅ **あなたの蓄電池は ~7.2 MWh**
   - 最大出力 1968 kW は瞬時能力
   - 基準値 1998 kW は調整力容量
   - 回帰式の係数0.040635と整合 ✅
